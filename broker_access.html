🔐 GitHub Personal Access Token (one-time setup)
Pour pouvoir sauvegarder direct sur GitHub depuis ici, tu dois créer un PAT GitHub avec les droits Contents: Read & Write sur ce repo.
  1. Va sur github.com/settings/personal-access-tokens/new
  2. Resource owner : ton compte. Repository access : Only select repositories → Bencode92/stock-analysis-platform
  3. Repository permissions : Contents → Read and write
  4. Generate token → copie-le et colle ci-dessous (stocké en localStorage, ne quitte pas ton navigateur)
