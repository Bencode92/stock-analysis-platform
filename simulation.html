Le saviez-vous ?

Le PER est l'un des rares placements qui offre à la fois :

  • Une déduction immédiate de votre revenu imposable (économie d'impôt)
  • La possibilité de débloquer les fonds pour l'achat de votre résidence principale

Idéal pour les personnes ayant une tranche marginale d'imposition (TMI) de 30% ou plus.

Économisez dès maintenant

Le PER vous permet de déduire vos versements de votre revenu imposable, ce qui réduit immédiatement votre impôt sur le revenu.

  • Déduction jusqu'à 10% de vos revenus professionnels
  • Plus votre TMI est élevée, plus l'économie est importante
  • Pour une TMI à 30%, versez 1000€ et économisez 300€ d'impôt

Financez votre résidence principale

Bien que normalement bloqué jusqu'à la retraite, le PER peut être débloqué dans plusieurs cas exceptionnels, notamment pour l'achat de votre résidence principale.

  • Déblocage autorisé pour l'achat de votre résidence principale
  • Permet de constituer un apport tout en réduisant vos impôts
  • Autres cas de déblocage : invalidité, décès, surendettement, etc.

Questions fréquentes

Comment estimer le rendement ?

Le rendement dépend de l'actif. Pour les indices boursiers, 7-10% est une estimation historique raisonnable sur le long terme.

Quelle durée d'investissement privilégier ?

Plus la durée est longue, plus l'effet des intérêts composés est puissant. Pour les actions, un horizon de 8-10 ans minimum est recommandé.

Quelle enveloppe choisir pour quel objectif ?

PEA pour les investissements en actions sur le long terme, Assurance-vie pour la flexibilité, PER pour la retraite.
